The Devanagari Converter

From transliterations to Devanagari and back.
This tool provides conversion from Latin transliterations to Devanagari and back.
You can input text in Harvard-Kyoto, IAST, ISO15919 and Devanagari Unicode.

It is based on the Haskell devanagari-transliterations library

An example: 'शुभ लाभ' is Sanskrit for 'Good Luck'. In the Harvard-Kyoto transliteration we can simply type 'zubha lAbha' into the input field.
As a result the text will be displayed in Devanagari in the respective output area.
You can as well enter a text in Devanagari and it will be displayed in the three transliterations.
The text in the output areas is updated while you type.

Harvard Kyoto transliteration – a brief tutorial

Harvard Kyoto-transliteration allows to type Sanskrit words with a normal computer keyboard. It is quite easy to learn, just remember the following rules:

  • Long vowels (marked by a dash over them in IAST) are always typed in upper case:
    Harvard-Kyoto IAST
    mahAbhArata mahābhārata
    bhagavadgItA bhagavadgītā
  • Upper case letters are also used to type letters that have a dot below them in IAST:
    Harvard-Kyoto IAST
    vATa vāṭa
    ISa īṣa
    ArUDha ārūḍha
    haMsa haṃsa
    bAlaH bālaḥ
    prANAyAma prāṇāyāma
    RSi ṛṣi
    amRta amṛta
  • The consonant ś is special. As 's' and 'S' are already used for 's' and 'ṣ' the letter 'z' is used:
    Harvard-Kyoto IAST
    zAstra śāstra
    zAnti śānti
  • Please also note the characters ñ – J and ṅ - G:
    Harvard-Kyoto IAST
    pataJjali patañjali
    aGga aṅga
  • Only for experts:
    standard Harvard-Kyoto does not support the anunasika (m̐ in IAST). With our converter you enter it as MM:
    Harvard-Kyoto IAST
    tAMMllokAn tām̐llokān
  • Special handling to avoid consonant ligatures (also not part of standard transliterations):
    Harvard-Kyoto Devanagari
    sadgamaye सद्गमये
    sad_gamaye सद्‌गमये
    vakSyAmi वक्ष्यामि
    vak\\SyAmi वक्‍‍ष्यामि
